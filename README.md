# EcoCart - Eco-Friendly eCommerce Frontend

**EcoCart** is a responsive, and user-friendly frontend for an eco-friendly eCommerce platform.

## Features

- Clean, sustainable design inspired by nature
- Fully functional shopping cart UI
- Product listing with filters and search
- Fully responsive design (mobile-first)
